# CryptEx
 This tool is a simple python3 script for the field of cryptography that allows encryption and decryption operations offline
 There are 6 encryption and decryption options inside. These are: aes,otp,rc4,rsa,blowfish and des encryption methods.
 After selecting the option you want, you can use this tool by filling in the input fields requested from you about the relevant encryption method,pleasant hackings...
 Before using this tool, make sure you have the pycryptodome library installed on your system!
 # Usage
 **pip3 install pycryptodome<br>git clone https://github.com/m4kr0-3Px/CryptEx.git<br>cd CryptEx<br>python3 CryptEx.py**
